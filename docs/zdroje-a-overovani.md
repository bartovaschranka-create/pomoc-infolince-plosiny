# Pravidla ověřování technických údajů

Pořadí důvěryhodnosti:
1. Oficiální technický list nebo návod výrobce pro konkrétní model a provedení.
2. Revizní protokol 2025 ke konkrétnímu stroji.
3. Produktová stránka Zeppelin CZ.
4. Ruční kontrola štítku konkrétního stroje.

Při rozporu zdrojů se údaj nesmí automaticky přepsat bez poznámky. V katalogu se uvede zdroj, datum ověření, varianta stroje a poznámka k rozdílu.
